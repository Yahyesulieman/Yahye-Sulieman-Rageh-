"# 📊 Mapping Administrative Regions of Ethiopia in R

## 📌 Overview
This project demonstrates spatial data analysis and GIS mapping using R. It focuses on visualizing administrative regions of Ethiopia from shapefiles and producing clear, publication-ready maps.

## 🧭 Objectives
- Map Admin Level 1 regions (Regional States)
- Filter and visualize the 13 official regions of Ethiopia
- Create a detailed map of Somali Region zones

## 🧰 Tools & Libraries
- R
- sf
- dplyr
- ggplot2
